Git and GitHub Commands

Git

It is a version control system.

It allows:

  1. access to history
  2. to collaborate with other programmers
  3. git account can be mentioned in portfolio

Commands

  1. Configuration

    • git config --global user.name "username"
    • git config --global user.email "email address"
    • git config --list

  2. Clone Command

    git clone "HTTPS link"

    It clones/copy a repository from remote (github) to your local device (laptop/computer).

  3. Change Directory

    • cd "folder name"

      It allows you to enter a folder, in order to make changes.

    • cd ..

      It allows you to exit a folder.

  4. List Files

    • ls

      It lists down all files.

    • ls -hidden

      It shows even the hidden files e.g .git

  5. Status Command

    git status

    It shows status of your files (untracked, modified, unmodified, staged etc.)

  6. Add command

    • git add "file name"

      In order to add files individually (files are staged and ready to be commited).

    • git add .

      In order to add all files collectively.

  7. Commit Command

    git commit -m"some message"

    Message should be regarding the change you made.

  8. Push Command

    • git push origin main

      Push the changes you made on your local device to github repo.

    • git push -u origin main

      Push all the changes you made on your local device to github repo (it also allows you to type only0 'git push' for all futher pushes)

  9. Make a folder locally

    mkdir "folder name"

    It allows you to make a folder locally on a code editor.

  10. Initialization

    git init

    It makes a local folder, a git repo (initilizes an empty repo).

  11. Make a remote file local

    git remote add origin "HHTPS link"

    A file made on github can be opened locally on your code editor.

  12. Verification

    git remote -v

    It allows to verify if a file is successfully made local.

  13. Branch Commands

    • git branch

      It shows all the existing branches plus the branch you are currently on.

    • git branch -M main

      Change name of a folder to main.

    • git branch -d "branch name"

      It deletes the mentioned folder.

  14. Checkout Commands

    • git checkout "branch name"

      It enter the branch mentioned.

    • git checkout -b "branch name"

      It allows you to make a new branch.

  15. Merging Codes

    • git diff "branch name"

      It checks the differences between the current branch you are on and the branch mentioned.

    • git merge "branch name"

      It merges the codes of the current branch you are on and the branch mentioned.

  16. Pull Command

    git pull origin main

    Pull request on command line

  17. Undoing Staged Changes

    • git reset "file name"

      It undo the last add command of the mentioned file.

    • git reset

      It undo the last add command of all files.

  18. Undoing Committed Changes

    • git reset HEAD~1

      It undo last add and commit command.

    • git reset "hash"

      It undo the add and commit commands till the mentioned hash.

    • git reset --hard"hash"

      It undo the add and commit commands till the mentioned hash on github too.

  19. To check commits

    git log

    It shows all the commits that have been made.

  20. Fork

    It allows you to make a rough copy of someone else's code
